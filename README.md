# iPAS AI 初級 1550 筆名詞與練習題

這個資料夾是 GitHub Pages 發布用版本，已合併既有 1500 筆名詞速查內容與新增 50 題練習題。

## 檔案說明

- `index.html`: 完整可發布頁面，內含 1500 筆原始名詞卡片與新增 50 題練習題卡片。
- `questions.js`: 新增 50 題練習題來源資料。
- `.nojekyll`: 避免 GitHub Pages 用 Jekyll 處理靜態檔。
- `source/20260624-IPAS-AI-question-bank-code-source.docx`: 原始 Word 來源檔備份。

## 題庫內容

目前包含 1550 筆內容：

- 原始網頁內容：1500 筆名詞速查卡片。
- 新增練習題：50 題。
- 新增科目一練習題：25 題。
- 新增科目二練習題：25 題。

## 注意事項

新增 50 題為原創練習題，不是官方公告試題逐字轉載。官方歷屆公告試題與學習指引請以 iPAS 官網為準：

https://ipd.nat.gov.tw/ipas/certification/AIAP/learning-resources

## GitHub Pages 發布

1. 將本資料夾內容上傳到 GitHub repository。
2. 到 repository 的 `Settings > Pages`。
3. 選擇要發布的 branch 與資料夾。
4. 儲存後等待 GitHub Pages 產生網址。
